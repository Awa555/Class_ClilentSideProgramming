What is JavaScript?

About JavaScript

JavaScript can make websites more interactive, interesting, and user-friendly.

JavaScript is a script that can also be called a programming language that enables you to perform complex functions on a web page. When a web page not only presents static content but also provides things like instant content updates, map dynamics, drawing 2D/3D graphics, movie playback control...

This programming language we call scripting. There can be written in HTML and executed automatically when the page is loaded. Scripts exist and execute as plain text. They do not require special preparation or compilation to run.

What can JavaScript in the browser do?
  • Insert new HTML into the web page to modify the existing web content and style of the web page.
  • Responsive to user behaviour, mouse clicks or moves, keyboard clicks.
  • Send network requests to remote servers, download or upload files (so-called AJAX and COMET technology).
  • Get or modify cookies to ask questions and send messages to visitors.
What makes JavaScript different from the other?
  • Full integration with HTML/CSS.
  • Perform simple tasks using simple tools.
  • It supported by all mainstream browsers and is opened by default.
