Exercise 1: linking an external file

  1. Create a folder for you HTML and CSS file.
  2. Create a file called index.html
  3. Create a new file called style.css
  4. add a link to the file in the head of your HTML file
  5. Add the CSS rule below to the CSS file:

body {
background-color: yellow;
}

Exercise 2: changing the style of multiple elements

  1. Add some rules to your css file
  2. Change the font color and background color of different types of elements — try selecting links, paragraphs, and lists
  3. Change the fonts of your page — try changing the font sizes and styles for different elements
